Progetto

Il nostro progetto era quello di creare un automa sulla base del concetto della Macchina di Turing con gli elementi  A B C  in ingresso per la formazione delle sequenze. La sequenza da riconoscere e’  AACBB 

Un automa è un sistema dinamico discreto (nella scansione del tempo e nella descrizione del suo stato) e tempo-invariante (il sistema si comporta alla stessa maniera indipendentemente dall'istante di tempo in cui agisce). Quando l'automa si trova in un dato stato, esso può accettare solo un sottoinsieme dei simboli del suo alfabeto.

Insiemi

Siamo partiti quindi a definire i diversi insiemi e le diverse tabelle di transizione / trasformazione che rappresentano le caretteristiche e il comportamento dell'automa.

  • I = { lettera }
  • VI = { A, B, C }
  • O = { stringa }
  • VO = { AACBB, no }
  • S = { attesa, a, aa, aac, aacb }

Qui sono riportati i vari passaggi che l'automa effettua in base allo stato in cui si trova e l'ingresso ricevuto.

Per esempio se l'automa si trova nello stato aac e riceve in ingresso una c ritorna nello stato iniziale perche quell'ingresso non fa parte della stringa da rilevare

In quest'altra tabella invece mostra quale valore l'automa manderà in uscita in seguito allo stato in cui si trova e a l'ingresso ricevuto

Per esempio se l'automa si trova nello stato aac e riceve in ingresso una c l'automa mandera in output la stringa no perche l'ingresso inserito non permette di individuare la stringa.

UML

Per avere un idea piu chiara e consultabile abbiamo creato un modello UML per la trasformazione dell'automa

Sono riportati i 5 stati dell'automa (nodi) e le relative transizioni segnate dall'ingresso e dalle uscite secondo le tabelle costruite sopra.

Per esempio la transizione - A - NO che parte dal nodo ATTESA e va sul nodo A sta a indicare che è stato ricevuto A in ingresso e l'automa restituisce NO in uscita

Modello Calc

Il tutto e’ stato messo insieme e abbiamo creato un modello in calc in modo da poterne testare le funzionalita’.

Sono stati creati due fogli di calcolo. Uno per la descrizione dove sono state implementate la varie tabelle e la descrizione visiva in Uml. Un altro foglio invece mostra l'evoluzione dell'automa nel tempo in seguito all'ingresso ricevuto.

Siamo andati a prendere attraverso funzioni speciali (CERCA.ORIZZ, CERCA.VERT e INDIRETTO) i valori nelle tabelle della descrizione in base a cio che veniva fornito nel foglio dell'evoluzione.

Implementazione Microbit

Lo stesso programma è stato implementato su un dispositivo fisico: MicroBit. Una schedina programmabile, semplice e intuitiva da utilizzare che ci ha permesso di testare il funzionamento dell'automa

Il codice è stato ottimizato per rendere riconoscibile qualsiasi sequenza con un massimo di tre caratteri per la formazione della stringa (Limitati dalle funzionalita di microbit. In futuro sarà magari possibile estendere questo limite).

Funzionamento

Il funzionamento del programma è piuttosto semplice:

  • Nel tasto A è associata la prima lettera in questo caso “A”
  • Nel tasto b è associata la seconda lettera in questo caso “B”
  • Con il movimento è associata la terza lettere in questo caso “C”

Il tipo di output può essere a “numero” (come nell'immagine che seguira) ovvero viene identificato il numero di volte che la sequenza viene riconosciuta all'interno dell'Array.

oppure può essere a “stringa” (come nell'immagine che seguira) ovvero viene mostrato a schermo l'intero array cioè l'intera stringa e si mostra con l'avanzamento dei caratteri una faccina sorridente a stringa indetificata

Implementazione in C

Funzionamento

Il programma ti chiederà di volta in volta di inserire un carattere, il quale verrà inserito in un array che verrà a sua volta esaminato per trovare la stringa rischiesta. Se l'esito è positivo verra stampata la stringa “Sequence found!”

L'array dove verranno salvati tutti i caratteri sarà della stessa dimensione della stringa da trovare di conseguenza piu la stringa sarà lunga e più memoria verrà occupata.

Codice e spiegazione

Di seguito è riportato il codice sorgente

Nella variabile costante STRING è riportata la stringa da controllare nella sequenza di input

Nella variabile STRING_LENGTH è definita la lunghezza della stringa (Attenzione questa costante deve sempre essere definita e deve corrispondere alla lunghezza della stringa)

Nel metodo main() vengono richiesti di volta in volta i parametri e pushati all'interno dell'array con la relativa funzione pushArray(char) la quale shifta di 1 l'array a sinistra

La funzione resetArray() pulisce l'array e viene utilizzato per inizializzarlo

La funzione checkArray() controlla la stringa all'interno dell'array e ne stampa l'esito

/**
    * AUTHOR: Condrache Marco Mihai
    * CLASS: 3^A 
    * DATE: 28/11/2019
    * DESCRIPTION: Program for the automaton based on the concept of turing machine
    * ? GITHUB: https: //github.com/MarcoMihaiCondrache
*/

#include <stdio.h>
#include <string.h>
#define STRING "aacbb" // STRING that will be searched into the array
#define STRING_LENGTH 5

// Declaring variables
char word[STRING_LENGTH];
void resetArray();
void pushArray(char);
void checkArray();

int main(int argc, char const *argv[])
{
    char current = STRING[0];
    resetArray();
    // ask for the chars
    while (strchr(STRING, current) != NULL)
    {
        printf("Insert char: ");
        scanf("%1s", &current);
        pushArray(current);
        checkArray();
    }
}

/**
 * Method that reset array to * values
 * @return void
 */
void resetArray()
{
    for (int i = 0; i < STRING_LENGTH; i++)
    {
        word[i] = '*';
    }
}
/**
 * Method that shift by 1 the array and append the value
 * @param char The value to append
 * @return void
 */
void pushArray(char value)
{
    for (int i = 0; i < STRING_LENGTH - 1; i++)
    {
        word[i] = word[i + 1];
    }
    word[STRING_LENGTH - 1] = value;
}
/**
 * Check if the array containts the STRING value
 * @return void
 */
void checkArray()
{
    if (strncmp(word, STRING, strlen(STRING)) == 0)
    {
        printf("Sequence found!\n");
    }
}

Implementazione in Java

Funzionamento

Il programma ti chiederà di volta in volta di inserire un carattere, il quale verrà inserito in un array che verrà a sua volta esaminato per trovare la stringa rischiesta. Se l'esito è positivo verra stampata la stringa “Sequence found!”

Codice e spiegazione

Di seguito è riportato il codice sorgente

La classe TuringAutomaton è la classe dove sono definiti i metodi per il funzionamento dell'automa Il metodo main contiene l'inzio del programma quindi deve essere richiamato per inizializzare il programma e’ di tipo statico e pubblico

Nella variabile costante STRING_TO_FIND è riportata la stringa da controllare nella sequenza di input

Il metodo pushInArray viene richiamato per inserire il carattere all'interno dell'array secondo il concetto LIFO

Il metodo checkArray viene richiamato per controllare nell'array la stringa e ne restituisce l'esito

import java.util.ArrayList;
import java.util.Scanner;

/**
 * Program based on the Turing Machine concept (console)
 * <br>
 * https://github.com/MarcoMihaiCondrache/TuringAutomaton
 *
 * @author  Condrache Marco Mihai
 * @version 1.0
 * @since   2019-11-28
 */

public class TuringAutomaton {
    /**
     * The final string that contains the string to find
     */
    private static final String STRING_TO_FIND = "aacbb";

    public static void main(String[] args) {
        String current;
        // full word to compare
        ArrayList<String> word = new ArrayList<String>();
        for (int i=0; i<STRING_TO_FIND.length(); i++){
            word.add("");
        }
        do {
            // insert and check the word with the new char
            Scanner reader = new Scanner(System.in);
            System.out.print("Insert char: ");
            current = reader.nextLine();
            word = pushInArray(word, current);
            if (checkWord(word)){
                System.out.println("Sequence found!\n");
            }
        }while (STRING_TO_FIND.contains(current));
        System.out.println(current);
    }

    /**
     * Method to check if the list contains
     * the STRING_TO_FIND and return true or false.
     *
     * @param  list The array to join to compare
     * @return  if the string contains the cost
     * @since 1.0
     */
    private static boolean checkWord(ArrayList<String> list){
        StringBuilder sb = new StringBuilder();
        for (String s : list)
        {
            sb.append(s);
        }
        return sb.toString().contains(STRING_TO_FIND);
    }

    /***
     * Method to push into array using LIFO concept
     *
     * @param list The array to push in
     * @param charT The char to insert
     * @return return the complete list
     */
    private static ArrayList<String> pushInArray(ArrayList<String> list, String charT){
        list.remove(0);
        list.add(charT);
        return list;
    }
}

Di seguito è riportato il codice sorgente

Nella variabile costante STRING è riportata la stringa da controllare nella sequenza di input

Vengono richiesti di volta in volta i parametri e pushati all'interno dell'array con la relativa funzione push_array(char) la quale shifta di 1 l'array a sinistra

"""
    * AUTHOR: Condrache Marco Mihai
    * CLASS: 3^A
    * DATE: 28/11/2019
    * DESCRIPTION: Program based on the Turing Machine concept (console)
    ? GITHUB: https://github.com/MarcoMihaiCondrache/TuringAutomaton
"""

# Define STRING variable to find inside characters
STRING = "aacbb"
# Define word variable that will contain the insert string
word = [""] * len(STRING)
# Define current variable that will contain the current char
current = STRING[0]


def push_array(char):
    """
    Push into array with Lifo concept

    :param char: The char to insert into the array
    :return:
    """
    word.pop(0)
    word.append(char)


def check_array():
    """
    Check if the array contains the sequence

    :return bool: Return the condition
    """
    return STRING in "".join(word)


while current in STRING:
    # Ask the char
    current = str(input("Insert char: "))
    # Insert it into the array
    push_array(current)
    # Check and eventually clear the array
    if check_array():
        print("Sequence found")

# If the while statement exits means that it has received an invalid char
print("Invalid char")

import 'dart:io';

/// Constant string that have the value to find inside the array
/// Type: [String]
const STRING = "aacbb";

/// Main method that request the char every time
/// Return: [void]
void main() {
  /// Generate a list with empty space to prevent [Segmentation Fault]
  List<String> word = List.generate(STRING.length, (index) => "");

  /// First char of the string
  String current = STRING.split("")[0];
  do {
    print("Insert char: ");
    current = stdin.readLineSync();

    /// Save chars with the LIFO concept
    word.removeAt(0);
    word.add(current);

    /// Check the string
    if (checkArray(word)) {
      print("Sequence found!");
    }
  } while (STRING.contains(current));
}

/// Method that check if the array contains the [STRING]
/// Param: [List<String> array]
/// Return: [bool]
bool checkArray(List<String> array) {
  return array.join("") == STRING ? true : false;
}
